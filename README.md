# Graphical user interface for RangeShifter v2.0

In this repository you can find the executables for the graphical user interface for RangeShifter v2.0.

You can also find the data needed to replicate the tutorials described in the [manual](https://github.com/RangeShifter/RangeShifter-software-and-documentation/blob/master/RangeShifter_v2.0_UserManual.pdf) under [Tutorials](./Tutorial_data)

Please note that this GUI is currently not maintained, and the executables are provided for users who want to run RangeShifter v2.0 with a GUI.

For users who want to use RangeShifter v3.0, please check one of the other interfaces (see [RangeShifter_batch](https://github.com/RangeShifter/RangeShifter_batch) or [RangeShiftR-pkg](https://github.com/RangeShifter/RangeShiftR-pkg)).
